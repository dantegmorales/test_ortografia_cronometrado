Test de Ortografía

Este test de ortografía está pensado para conocer tu nivel ortográfico. Apreciarás el dominio ortográfico del vocabulario más usual y tu capacidad para acentuar correctamente las palabras.

El test está diseñado en tres módulos y contarás con un tiempo límite para responder cada uno de ellos, te pedimos que leas atentamente las instrucciones ya que no hay opción de volver a preguntas anteriores.

PRIMER MÓDULO
